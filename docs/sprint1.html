ConwayLife Sprint1

Introduction

Realizzazione in Java del GAME OF LIFE DI CONWAY

Functional Requirements

  • R1: Implementare in Java una versione del gioco Game of Life di Conway, inteso come zero-player game. Il sistema deve modellare una Griglia composta da Celle il cui stato (viva o morta) cambia nel tempo secondo le regole definite dal Game of Life.
  • R2: L’utente deve poter:
    • definire la configurazione iniziale della griglia
    • osservare l’evoluzione temporale del gioco tramite una rappresentazione adeguata (si veda Problema della rappresentazione del gioco)
    • mettere in pausa e riavviare l’esecuzione
    • azzerare la griglia quando il gioco è in stato di pausa

Analisi dei Requisiti

  • Che cosa si intende per "Cella"?
  • interface ICell - Componente caratterizzato da uno stato booleano (vivo/morto).
    • void setStatus(boolean alive) - Imposta direttamente lo stato della cella.
    • boolean isAlive() - Restituisce true se la cella è viva, false altrimenti.
    • void toggleState() - Cambia lo stato corrente della cella (viva ↔ morta).
  • Che cosa si intende per "Griglia"?
  • interface IGrid - Struttura bidimensionale ordinata che contiene celle e fornisce operazioni di gestione.
    • int getRows() - Fornisce il numero totale di righe.
    • int getCols() - Fornisce il numero totale di colonne.
    • boolean isAlive(int row, int col) - Indica se la cella in posizione (row, col) è viva.
    • void setStatus(int row, int col, boolean status) - Modifica lo stato della cella indicata.
    • int countAliveNeighbors(int row, int col) - Calcola il numero di celle vive adiacenti alla posizione data.
    • ICell getCell(int row, int col) - Restituisce la cella corrispondente alla posizione specificata.
  • Che cosa si intende per "Game of Life"?
  • interface ILife - Componente responsabile della gestione dell’evoluzione applicando le regole del gioco.
    • void computeNextGeneration() - Determina e applica la generazione successiva.
    • boolean isAlive(int row, int col) - Restituisce lo stato della cella indicata.
    • void setCell(int row, int col, boolean alive) - Definisce lo stato iniziale o corrente di una cella.
    • int getRows() - Restituisce il numero di righe della griglia associata.
    • int getCols() - Restituisce il numero di colonne della griglia associata.
