ConwayLife Sprint1

Introduction

Realizzazione in Java del GAME OF LIFE DI CONWAY

Requirements

  • R1: Sviluppare in Java una versione del gioco Life di Conway, concepita come zero-player game. Il sistema deve prevedere una Griglia di Celle il cui stato (cella "viva" o cella "morta") cambia nel tempo secondo le regole definite dal ConwayLife
  • R2: L'utente deve poter:
    • definire la configurazione iniziale della griglia di gioco
    • osservare l'evoluzione del gioco in una modalità adeguata (si veda Problema della vista del gioco)
    • interrompere e riavviare l'evoluzione del gioco
    • azzerare (quando il gioco è fermo) la configurazione corrente della griglia

Requirement analysis

  • Cosa si intende per "Cella"?
  • interface ICell - Entità che possiede uno stato binario: vivo o morto.
    • void setStatus(boolean alive) - Imposta esplicitamente lo stato della cella.
    • boolean isAlive() - Restituisce true se la cella è viva, false in caso contrario.
    • void switchCellState() - Alterna lo stato attuale della cella (viva ↔ morta).
  • Cosa si intende per "Griglia"?
  • interface IGrid - Entità che rappresenta una struttura bidimensionale ordinata di celle e gestisce le operazioni su di esse.
    • int getRows() - Restituisce il numero totale di righe della griglia.
    • int getCols() - Restituisce il numero totale di colonne della griglia.
    • boolean isAlive(int row, int col) - Verifica se la cella nella posizione (row, col) è viva.
    • void setStatus(int row, int col, boolean status) - Modifica lo stato della cella nella posizione (row, col).
    • int countNeighborsLive(int row, int col) - Calcola il numero di celle adiacenti vive rispetto alla posizione (row, col).
    • ICell getCell(int row, int col) - Restituisce il riferimento alla cella situata in (row, col).
  • Cosa si intende per "Gioco Life di Conway"?
  • interface ILife - Entità responsabile della gestione dell'evoluzione del gioco applicando le regole di Conway.
    • void nextGeneration() - Determina e applica lo stato della generazione successiva.
    • boolean isAlive(int row, int col) - Controlla lo stato della cella nella posizione (row, col).
    • void setCell(int row, int col, boolean alive) - Imposta lo stato iniziale o attuale di una cella.
    • int getRows() - Restituisce il numero di righe della griglia gestita.
    • int getCols() - Restituisce il numero di colonne della griglia gestita.
